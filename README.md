# JTIKS - Premium Beauty Brand Website

A premium beauty brand website built with Astro, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework**: Astro (SSR/Hybrid mode)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Hosting**: Hostinger shared hosting

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

- `/src/pages` - Astro pages/routes
- `/src/components` - Reusable components
- `/src/layouts` - Page layouts
- `/src/data` - Dynamic content data
- `/public` - Static assets
